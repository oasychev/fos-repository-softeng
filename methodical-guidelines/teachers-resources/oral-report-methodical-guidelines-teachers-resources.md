### 8. РЕКОМЕНДАЦИИ ПО ИСПОЛЬЗОВАНИЮ РЕСУРСОВ

#### 8.1. При подготовке к лекциям

- Используйте **официальную документацию** (Git, PostgreSQL, Python, Docker) как основной источник достоверной информации.
- Обращайтесь к **Pro Git** для углублённого изучения Git-команд, особенно разделы по rebase, merge, работе с историей.
- Используйте **Atlassian Git Tutorials** для наглядных объяснений сложных концепций (merge vs rebase, разрешение конфликтов).

#### 8.2. При проверке устных докладов

- Сверяйте информацию студентов с **официальной документацией** соответствующих инструментов.
- Для оценки раздела «случаи использования» используйте **неофициальные источники** (блоги, Atlassian, Stack Overflow, GitHub Discussions) — именно там часто описываются реальные практические сценарии.
- Задавайте вопросы, требующие понимания, а не простого воспроизведения документации.

#### 8.3. При подготовке лабораторных работ

- Используйте **NumPy User Guide** и **Pandas User Guide** как основу для заданий по обработке данных.
- Применяйте **FastAPI Tutorial** как пошаговое руководство для создания API.
- Используйте **Docker Getting Started** для вводных заданий по контейнеризации.
