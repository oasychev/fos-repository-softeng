# Лабораторное занятие 8: Критерии оценивания

**Максимальная оценка за выполнение лабораторной работы составляет 7 баллов.**

Итоговая оценка определяется на основании суммарного процента выполнения всех этапов лабораторной работы с последующим пропорциональным переводом в балльную систему (7 баллов соответствует 100% выполнения).

Оценка выставляется по результатам выполнения основных этапов лабораторной работы в соответствии с критериями, представленными в таблице.

| Этап лабораторной работы                              | Критерии выполнения                                                                                                                                                                     | Максимальная доля оценки |
| ----------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -----------------------: |
| **1. Контейнеризация приложения**                     | Подготовлены Dockerfile и конфигурация Docker Compose, настроены необходимые сервисы, зависимости, сети и переменные окружения. Контейнеры успешно собираются и запускаются.            |                  **35%** |
| **2. Локальное тестирование и развертывание системы** | Выполнен запуск контейнеризированного приложения, проверена работоспособность компонентов системы, выполнено развертывание на виртуальной машине и подтверждена доступность приложения. |                  **30%** |
| **3. Настройка CI/CD через GitHub Actions**           | Создан workflow GitHub Actions, настроен автоматический запуск проверок, выполнены проверки проекта, настроена защита основной ветви репозитория.                                       |                  **25%** |
| **4. Документирование результатов**                   | Подготовлена инструкция по развертыванию системы, описаны особенности контейнеризации и CI/CD-пайплайна, представлены результаты проверки работоспособности.                            |                  **10%** |

> **Примечание.** Каждый этап лабораторной работы оценивается независимо. Итоговая оценка определяется суммой набранных процентов по всем этапам с последующим переводом в баллы.
