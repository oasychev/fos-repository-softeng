# Модуль 4. Инструменты промышленной разработки на Python

## Назначение модуля

Модуль посвящен освоению современного инструментария промышленной разработки на Python, необходимого для создания поддерживаемых, тестируемых и готовых к развертыванию ИИ-систем. Студенты изучают управление зависимостями и структурой проекта через Poetry, автоматизацию контроля качества кода с pre-commit, написание модульных тестов с pytest, организацию логирования, строгую типизацию и документирование кода, а также управление конфигурациями с использованием Pydantic и python-dotenv. Особое внимание уделяется практическому применению всех инструментов в рамках сквозного проекта — от инициализации репозитория до готового к публикации пакета.

## Проверяемые результаты обучения

- PL-1.1 (Базовый) — Разрабатывает и отлаживает прикладные решения с использованием языка программирования Python, тестирует, испытывает и оценивает качество таких решений.
- LC-5.1 (Базовый) — Осуществляет выбор инструментов и инженерных практик промышленной разработки систем ИИ, развертывания и сопровождения моделей МО в продуктивной среде.

## Содержание модуля

### Тематические сущности

- Poetry — управление зависимостями и упаковка.
- Git + pre-commit — контроль версий и качество кода. 
- pytest + pytest-mock — тестирование. 
- logging — логирование. 
- Pydantic + python-dotenv — конфигурация и валидация. 
- Типизация (Type Hints) — качество кода. 
- Документирование (Docstrings) — качество кода. 
- PEP8 — стиль кода
