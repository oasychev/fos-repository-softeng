# Лабораторная работа. Анализ данных с Pandas и визуализация с Matplotlib

## Цель работы
Провести полный исследовательский анализ данных (EDA) датасета Titanic с бизнес-выводами. Студент выступает в роли аналитика страховой компании.

## Задание

### Часть 1. Загрузка и первичный анализ данных

1. Загрузить данные из CSV-файла с оптимизацией памяти (параметры dtype, usecols).
2. Выполнить первичный просмотр: head, tail, info, describe.
3. Определить числовые и категориальные признаки, выявить пропуски.

### Часть 2. Очистка и подготовка данных

1. Обработать пропуски:
   - Age — медиана.
   - Embarked — мода.
   - Удалить неинформативный столбец Cabin.
2. Проверить и удалить дубликаты.
3. Преобразовать категориальные признаки в формат для анализа.

### Часть 3. Проверка гипотез

1. Сформулировать 3–4 гипотезы о факторах, влияющих на выживаемость (например, женщины выживают чаще мужчин, пассажиры 1-го класса выживают чаще).
2. Проверить гипотезы с помощью группировок (groupby) и визуализаций (столбчатые диаграммы).
3. Провести статистическую проверку:
   - t-тест для числовых признаков (scipy.stats.ttest_ind).
   - Критерий χ² для категориальных (scipy.stats.chi2_contingency).
   - Интерпретировать p-value.

### Часть 4. Анализ "портрета" пассажира

1. Определить группы с наименьшей и наибольшей выживаемостью.
2. Построить сводную таблицу выживаемости по полу и классу (pivot_table).
3. Рассчитать дополнительные метрики: средний возраст и стоимость билета выживших и погибших.

### Часть 5. Визуализация для не-технической аудитории

Построить набор графиков, понятных менеджеру:
1. Столбчатая диаграмма выживаемости по полу.
2. Столбчатая диаграмма выживаемости по классу.
3. Сгруппированная диаграмма по полу и классу.
4. Boxplot возраста выживших и погибших.
5. Boxplot стоимости билета выживших и погибших.
6. Тепловая карта корреляций (sns.heatmap).

### Часть 6. Подготовка данных для ML (дополнительно)

1. Кодирование категориальных признаков (pd.get_dummies).
2. Масштабирование числовых признаков (StandardScaler).
3. Создание пайплайна предобработки с sklearn.pipeline.Pipeline.
