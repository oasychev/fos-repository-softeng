# Модуль 6. Анализ данных с Pandas и визуализация с Matplotlib

## Назначение модуля
Модуль формирует навыки исследовательского анализа данных (EDA) с использованием библиотек Pandas, Matplotlib и Seaborn. Студенты осваивают загрузку, очистку, трансформацию, визуализацию данных и подготовку датасетов для задач машинного обучения.

## Проверяемые результаты обучения
- **PL-1.1 (Средний)** — Разрабатывает и отлаживает прикладные решения с использованием Python, тестирует и оценивает качество.
- **PL-1.2 (Средний)** — Осуществляет выбор инструментов Python для обработки данных, ML и визуализации с заданными требованиями.
- **BD-2.1 (Базовый)** — Определяет требования к наборам и качеству данных для решения задач машинного обучения.
- **BD-1.2 (Базовый)** — Применяет методы предварительного анализа данных, включая визуализацию.

## Содержание модуля
### Тематические сущности
- Библиотека Pandas: структуры Series и DataFrame, загрузка данных (read_csv, read_excel, read_json).
- Предварительный просмотр данных: head, tail, info, describe.
- Индексация и выборка данных: loc, iloc, at, iat.
- Фильтрация данных с использованием логических условий.
- Обработка пропусков: isnull, dropna, fillna.
- Обработка дубликатов: duplicated, drop_duplicates.
- Группировка и агрегация: groupby, agg, apply.
- Соединение таблиц: merge, concat, join, pivot_table, crosstab.
- Оптимизация памяти: параметры dtype, usecols, chunksize.
- Визуализация с Matplotlib и Seaborn: гистограммы, столбчатые диаграммы, boxplot, scatter plot, тепловая карта корреляций.
- Подготовка данных для ML: кодирование категориальных признаков (pd.get_dummies), масштабирование (StandardScaler), пайплайны предобработки (sklearn.pipeline.Pipeline).
- Параллельная обработка данных: swifter, modin (дополнительно).
