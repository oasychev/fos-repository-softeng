# Зачет с оценкой

### Система оценивания результатов освоения дисциплины

Итоговая оценка по курсу формируется накопительным принципом на основе балльно-рейтинговой системы. В течение семестра студент выполняет различные виды учебной деятельности, каждая из которых оценивается в баллах. Максимальная сумма баллов за все модули и курсовую работу может составлять более **100 баллов**.

#### 1. Структура накопления баллов

Учебный материал разделён на 8 модулей. В каждом модуле предусмотрены следующие оценочные мероприятия:

- **Лабораторная работа** – максимальная оценка **7 баллов** за каждый из 8 модулей (всего **56 баллов** за все лабораторные работы).  
  Лабораторные работы являются обязательными для допуска к итоговому контролю. Работа считается выполненной, если она сдана в установленный срок и соответствует минимальным требованиям, предъявляемым к результату.

- **Доклады** – в каждом модуле студент может выступить с устным докладом по теме модуля. Один доклад оценивается в **2 балла**. Количество докладов не ограничено, однако для получения высокой итоговой оценки требуется определённое число выступлений (см. шкалу перевода баллов). Максимальное количество баллов за доклады не лимитируется, но учитывается в общем рейтинге.

- **Групповая игра** – предусмотрена в модуле 3 («Продвинутый Git и инженерные практики») и оценивается в **4 балла**. Участие в игре является обязательным элементом для положительной аттестации.

- **Курсовая работа** – выполняется по итогам всех модулей и оценивается в **40 баллов**. Курсовая работа является обязательным видом контроля. Для получения оценки «удовлетворительно» необходимо набрать как минимум **1 балл** за курсовую работу.

#### 2. Шкала перевода баллов в итоговую оценку

Итоговая оценка выставляется на основе суммы всех набранных баллов при выполнении обязательных условий.

- **Оценка «УДОВЛЕТВОРИТЕЛЬНО» (3)**  
  Для получения этой оценки студенту необходимо:
  - выполнить **все** лабораторные работы (по каждому из 8 модулей);
  - принять участие в групповой игре;
  - выполнить курсовую работу с результатом **не менее 10 баллов**;
  - набрать суммарно **более 59 баллов** (т.е. 60 баллов и выше).  
  *Важно:* несоблюдение любого из перечисленных обязательных условий автоматически лишает права на положительную оценку, даже при наличии достаточного количества баллов.

- **Оценка «ХОРОШО» (4)**  
  Для получения оценки «хорошо» студент должен:
  - выполнить все условия для оценки «удовлетворительно»;
  - набрать суммарно **более 74 баллов** (т.е. 75 баллов и выше);
  - выступить с устным докладом **не менее одного раза** в течение семестра (в любом модуле).

- **Оценка «ОТЛИЧНО» (5)**  
  Для получения оценки «отлично» студент должен:
  - выполнить все условия для оценки «удовлетворительно»;
  - набрать суммарно **более 89 баллов** (т.е. 90 баллов и выше);
  - выступить с устными докладами **не менее трёх раз** в течение семестра (в любых модулях). Доклады должны быть сделаны в рамках разных модулей.

#### 3. Особые замечания

- Все баллы суммируются автоматически. Студент имеет возможность отслеживать свой прогресс в течение семестра.
- Доклады не являются обязательными для всех студентов, но служат эффективным инструментом для повышения итоговой оценки и развития навыков публичной презентации научно-технических результатов.
- Курсовая работа является сквозным проектом, интегрирующим знания из всех модулей. Её выполнение обязательно для всех студентов, претендующих на любую положительную оценку.

**Резюмирующее правило:**  
Итоговая оценка зависит не только от суммы баллов, но и от выполнения обязательных активностей (лабораторные, игра, курсовая) и количества публичных выступлений. Рекомендуется планировать свою работу так, чтобы равномерно распределять усилия в течение семестра и своевременно участвовать в докладах для повышения рейтинга.
