# Зачет

Зачет проводится на основании балльно-рейтинговой системы в форме дифферинцированного зачета. Баллы, набранные в течение семестра за компоненты курса, переводятся в итоговую академическую оценку в соответствии с рейтинговой шкалой:

| Оценка | Баллы  |
|-------:|:------:|
| неуд.  | 0-59   |
| удовл. | 60-74  |
| хор.   | 75-89  |
| отл.   | 90-100 |

> [!IMPORTANT]
> Минимальный пороговый балл для успешной сдачи дифференцированного зачета составляет 60 баллов.
