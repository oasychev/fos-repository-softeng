# Модуль 5. Введение в Data Science. Библиотеки для работы с данными (NumPy)

## Назначение модуля
Модуль знакомит с экосистемой Python для Data Science и формирует фундаментальные навыки работы с библиотекой NumPy. Студенты осваивают векторизованные вычисления, оптимизацию производительности и реализацию базовых алгоритмов машинного обучения с нуля.

## Проверяемые результаты обучения
- **PL-1.1 (Средний)** — Разрабатывает и отлаживает прикладные решения разной сложности с использованием Python, тестирует и оценивает качество.
- **PL-1.2 (Средний)** — Осуществляет выбор инструментов Python для обработки данных, ML и визуализации с заданными требованиями.
- **ML-3.1 (Средний)** — Обосновывает применение классических методов МО с пониманием их математической сущности.
- **PL-1.3 (Базовый)** — Разрабатывает и поддерживает системы обработки больших данных различной степени сложности.

## Содержание модуля
### Тематические сущности
- Экосистема Python для Data Science: NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook.
- Библиотека NumPy: объект ndarray, создание массивов, индексация и срезы.
- Векторизованные операции и универсальные функции (ufunc).
- Broadcasting (правила расширения массивов).
- Агрегатные функции и редукция по осям.
- Операции линейной алгебры.
- Оптимизация вычислений: профилирование кода (%timeit), выбор оптимальных методов.
- Многопоточная обработка данных в NumPy (threading, concurrent.futures).
- Сравнение производительности: реализация алгоритмов на чистом Python vs NumPy vs Scikit-learn.
- Работа с данными, не помещающимися в память (np.memmap).
