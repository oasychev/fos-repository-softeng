# Доклад. Введение в Data Science. NumPy

## Формат
Устный доклад 5–7 минут на практическом занятии.

## Список тем
1. Векторизованные операции в NumPy: принципы и преимущества перед циклами Python.
2. Broadcasting в NumPy: правила расширения массивов для арифметических операций.
3. Сравнение производительности: реализация алгоритмов на чистом Python vs NumPy vs Scikit-learn.
4. Оптимизация вычислений в NumPy: профилирование и выбор оптимальных методов.
5. Работа с большими данными в NumPy: np.memmap и обработка данных, не помещающихся в память.
6. Многопоточная обработка данных с использованием NumPy и threading.
