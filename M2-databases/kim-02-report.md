# Доклад. Базы данных и SQL

## Формат
Устный доклад 5–7 минут на практическом занятии.

## Список тем
1. Нормализация баз данных: 1NF, 2NF, 3NF на практическом примере (на основе датасета Titanic).
2. Соединения таблиц в SQL: INNER, LEFT, RIGHT, FULL JOIN и их применение в аналитических задачах.
3. Оптимизация запросов: EXPLAIN ANALYZE и индексы в PostgreSQL.
4. Оконные функции в SQL: ROW_NUMBER, RANK, LAG, LEAD для аналитики.
5. Транзакции в PostgreSQL: ACID, уровни изоляции, практическое применение.
6. Многопоточная обработка данных при интеграции Python и PostgreSQL: преимущества и подводные камни.
7. Сравнение SQLite, PostgreSQL и MySQL для ИИ-проектов.

## Требования к докладу
- Раскрыть теоретические аспекты темы.
- Привести практические примеры использования.
- Подготовить презентацию (опционально).
