# Модуль 2. Базы данных и SQL

## Назначение модуля
Модуль посвящен практическому применению реляционных баз данных в ИИ-проектах. Студенты осваивают проектирование схем БД, написание сложных аналитических запросов, оптимизацию производительности, работу с транзакциями и интеграцию с Python с использованием многопоточности для повышения производительности при загрузке и выгрузке данных.

## Проверяемые результаты обучения
- **PL-1.1 (Средний)** — Разрабатывает и отлаживает прикладные решения с использованием Python, тестирует и оценивает качество.
- **PL-1.2 (Средний)** — Осуществляет выбор инструментов Python для обработки данных, ML и визуализации с заданными требованиями.
- **BD-3.1 (Базовый)** — Создает базы данных в реляционных СУБД, заполняет данными, пишет SQL-запросы.
- **PL-1.3 (Базовый)** — Разрабатывает и поддерживает системы обработки больших данных различной степени сложности.

## Содержание модуля
### Тематические сущности
- Реляционная модель данных: сущности, атрибуты, связи, первичные и внешние ключи.
- Нормализация (1NF, 2NF, 3NF).
- PostgreSQL: установка, настройка, работа через psql.
- Язык SQL: DDL (CREATE TABLE, ALTER TABLE, DROP TABLE), DML (INSERT, UPDATE, DELETE), SELECT, WHERE, ORDER BY.
- Агрегатные функции, группировка GROUP BY, фильтрация групп HAVING.
- Соединение таблиц: INNER JOIN, LEFT JOIN, RIGHT JOIN.
- Оконные функции: ROW_NUMBER, RANK, LAG, LEAD.
- Подзапросы.
- Индексы и оптимизация запросов (EXPLAIN ANALYZE).
- Транзакции (BEGIN, COMMIT, ROLLBACK).
- Интеграция с Python: psycopg2 (синхронный), asyncpg (асинхронный), пул соединений.
- ETL-пайплайны: загрузка данных из CSV с использованием COPY.
- Многопоточная загрузка и выгрузка данных с использованием ThreadPoolExecutor.
